# Microsoft Rewards Tool

**Ferramenta completa e bonita para Microsoft Rewards — Feita especialmente para caçadores brasileiros.**

Uma ferramenta leve, moderna e totalmente funcional para acompanhar seus pontos diários, streak, meta mensal e conversão em reais.

## ✨ Funcionalidades

### Principais recursos
- **Meta Mensal** com barra de progresso em tempo real (salva automaticamente)
- **Sequência de Dias (Streak)** com edição manual e animação de milestone especial (a partir de 44 dias + a cada 10 dias)
- **Conversor Pontos → Reais** em tempo real com taxa personalizável
- **Taxa de conversão** dinâmica baseada no seu exemplo (ex: 5.165 pts = R$ 30,00)
- **Histórico completo** de pontos com data, valor em reais e total acumulado
- Adicionar/editar/excluir itens do histórico
- **Estimativa inteligente** de dias para bater a meta (só aparece a partir do 7º dia registrado)
- **Tempo restante no mês** (mostra dias ou horas/minutos no final do mês)
- **Tooltips explicativos** em todos os cards (modal bonito com informações úteis)
- **Exportar** dados completos em JSON
- **Importar** backup completo via JSON
- histórico exportável também em CSV (visualizável em Excel, Google Sheets)
- **Tema Dark/Light** com persistência
- **Modo Horizontal (Landscape)** otimizado para desktop/tablet
- Layout **full-bleed** (cards estendidos até as bordas sem fendas laterais)
- Totalmente responsivo e com animações suaves

### Dados salvos automaticamente
Tudo fica guardado no `localStorage` do seu navegador: meta, streak, histórico, exemplo de conversão e preferência de tema.

## 🛠 Tecnologias

- **HTML5 + CSS3** (Flexbox, variáveis CSS, animações)
- **Vanilla JavaScript** (sem frameworks)
- Single-file (apenas um arquivo `.html`)

## 🚀 Como usar

1. Baixe o arquivo **`Index.html`**
2. Abra diretamente no navegador (recomendado: Edge ou Chrome, não testado:Firefox)
3. Comece a registrar seus pontos diários — tudo é salvo automaticamente

## 📌 Observações

- A ferramenta usa **apenas os dias que você registra** para calcular a média diária.
- Quanto mais consistente você for em adicionar os pontos, mais precisa fica a projeção da meta.
- O streak aumenta automaticamente toda vez que você adiciona pontos do dia.
